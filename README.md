# 📚 Digital Library Management System

A professional Library Management application developed using **Python** and **Streamlit**. This project demonstrates the core concepts of **Object-Oriented Programming (OOP)** like Classes, Objects, and State Management.

## ✨ Features
* **Add Books:** Register new books with title, author, and quantity.
* **Search:** Find books instantly by title or author.
* **Borrow/Return:** Functional system to manage book availability.
* **Real-time UI:** Interactive dashboard built with Streamlit.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Frontend:** Streamlit
* **Version Control:** Git & GitHub

## 🚀 How to Run
1. Clone the repository.
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `streamlit run main.py`
